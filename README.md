## Криминальная Молдова

Исходная версия пятой части для ознакомления.

Для запуска проекта требуются плагины:  
- Video Copilot - Element 3D 2.2.3 (build 2192)
- Video Copilot - Saber v1.0.40
- Video Copilot - Twitch v1.1
- Ogg Plug-in

Шрифты:  
- Eurostile-Demi
